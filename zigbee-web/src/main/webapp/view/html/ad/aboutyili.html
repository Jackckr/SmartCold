    逸励实业（上海）有限公司创立于国家十三五规划的开局之年，是一家中新合资企业，专注于制冷行业新技术、新产品的研发、制造、推广及应用。集贸易、制造、工程服务、投资为一体，致力于打造成冷链行业发展的旗舰。

    逸励实业整合了国内外非常优秀的团队，每个团队各自经过多年布局，建成了自己完善的冷链服务体系。整合后的逸励，总部设在上海，服务网络覆盖了上海、北京、广州、深圳、南宁、昆明、呼和浩特、长春、沈阳、银川、无锡、海口、香港，东南亚地区包括新加坡、马来西亚、印尼等国家都有办事机构。工程技术及服务人员达到220多人。整个网络为沃尔玛、家乐福、大润发、华润万家、百佳等大型卖场提供及时周到的工程安装及维修保养；同时公司承接各大中小型冷库工程，从前期规划、设计到施工、验收交付使用。涉及领域包括冷链物流、食品速冻与加工、医药、果蔬气调、菌菇培养、环境气候模拟实验室等。网络成员此前分别为康新物流、普洛斯、东方航空食品配餐中心、丰驰物流、波隆冷链、恒伟物流、润恒物流、新发地批发市场、南京众彩物流等大型物流冷库项目提供冷库产品和工程服务；为艾莱发喜食品、安井食品、宁波申洲、厦门联华食品、棒约翰中央厨房、内蒙科尔沁、蒙都肉业、小肥羊、清华大学中央厨房、海丰渔业等大型食品加工型冷库提供冷库产品及工程服务；为康美药业、东宝药业、金马药业、紫鑫药业、帝斯曼药厂、兴齐眼药厂、东阳光、苏州国药、江西药械厂等制药企业提供冷库产品及工程服务；在农产品气调保鲜领域，为沃田农业、众仁果业、榕福气调库、万吉冷藏、平乐气调库等项目提供制冷、气调产品及工程服务。

    逸励实业供链业务板块，代理经销各种制冷产品配件、耗材、代理BITZER、PANASONIC、YORK、CARRIER、HITACHI等品牌压缩机及机组；代理经销EDEN、PANASONIC、GUNTER、ECO、GEA等品牌的蒸发器和冷凝器；代理经销DANFOSS、PARKER、PARKER、EMERSON等品牌的阀件及电气控制。
