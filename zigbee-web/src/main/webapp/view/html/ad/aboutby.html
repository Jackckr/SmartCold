    常州市贝源冷冻设备有限公司是由无锡倍源冷冻设备有限公司与逸励实业（上海）有限公司就冷库板业务进行整合，并注入资金，投入全新的自动化发泡设备，建设的一个现代化聚氨酯夹芯板加工厂。生产能力从年产1 5万平方扩到年产50万平方。 新工厂位于江苏省常州市武进开发区，总投资1000万。 生产50mm、75mm、1OOmm、120mm、150mm、200mm厚度的聚氨酯冷库板，面板可以按照用户的需求制作彩钢板、不锈钢板、压花铝板、花纹钢板等。板材连接型式包括偏心钩与非偏心钩连接。公司从德国引进先进的生产技术， 向市场提供各种高端的冷库门，包括手动平移门、电动平移门、铰链门、车间用防撞自由门等。

    公司产品广泛应用于商超、食品工厂、冷链物流、医药仓储及工厂、环境模拟室、农产品保鲜等领域。 从2013年投向市场以来受到广泛好评。
